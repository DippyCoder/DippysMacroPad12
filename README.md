# DIPPYS MACRO PAD 12

**Mini hot-swappable mechanical keyboard with 12 keys, USB-C, RP2040 microcontroller, mini OLED display, backlighting and front lighting.**

Dippys Macro Pad 12 is a fork of ANAVI Macro Pad 12 and is an open source, custom-programmable, mini mechanical keyboard/keypad.
Original repository: https://github.com/AnaviTechnology/anavi-macro-pad-12/

# Install firmware

1. Press "R" Button
2. Hold "B" Button and press the "R" Button, then release
3. Open the new Removable Drive
4. Download [CIRCUITPYHON](https://circuitpython.org/board/seeeduino_xiao_rp2040/) and drag it into the Removable Drive
5. When rebooted, check if the Removable Drive is called "CIRCUITPYTHON"
6. Add the files from the "firmware" folder of this repository into the "CIRCUITPYTHON" drive. (Replace the files that already exist)
7. Verify if the Keyboard boots.
